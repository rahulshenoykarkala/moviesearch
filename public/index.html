Search API

Movie Search API

Online REST API for Testing and Prototyping of movie Search/catalog application.

What's in the store?
Catalog

Movie List with Attributes

Catalog of 60 movies with title, description, year of release, genre, total duration of movie, directors. In addition, the API also provides the movie ratings from IMDB, Rotten Tomatoes and Metacrirtic.

Go to Specs

Search

Search and Filter

Search for movies either with a keyword or filter or a combination of both. Keyword search is done on all movie attributes. Further, the filtering can be done with attributes year, director and genre. Other attribute filterings are not optimized and may not work as expected.
